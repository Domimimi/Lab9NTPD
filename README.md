Wnioski:

- DataFrame: dane są zorganizowane w tabelach, możliwe są zapisy danych do plików CSV czy parquet

- RDD: dane są w formie kolekcji elementów przetwarzanych linia po linii, wymaga ręcznego dzielenia tekstu i rzutowania typów

- Do codziennej pracy lepszym rozwiązaniem jest DataFrame

- RDD warto używać, gdy dane są niestandardowe
